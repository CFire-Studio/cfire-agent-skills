# 参考配置模板

此目录包含艺人档案配置文件的通用模板。

## 使用说明

1. 复制此目录为新艺人配置目录
2. 按照 BOOTSTRAP.md 的引导流程填写各配置文件
3. 删除或重命名模板文件，填写真实内容

## 模板文件说明

### 稳定层（系统配置，不建议修改）
- PROFILE_SCHEMA.md - 档案结构规范
- BOOTSTRAP.md - 新艺人引导流程
- LEARNING.md - 学习循环机制
- AGENTS.md - 运行准则
- BOUNDARIES.md - 安全边界

### 上下文层（艺人配置，谨慎修改）
- IDENTITY.md - 艺人身份（示例）
- SOUL.md - 性格与价值观（示例）
- GOALS.md - 目标与方向（示例）
- CONTENT_STRATEGY.md - 内容策略（示例）
- VOICE_STYLE.md - 表达风格（示例）
- FAN_RELATIONSHIP.md - 粉丝关系（示例）

### 易变层（运营数据，可频繁更新）
- MEMORY.md - 长期记忆（示例）
- REVIEW.md - 审查与复盘（示例）
- TOOLS.md - 工具协同（示例）
- HEARTBEAT.md - 心跳与主动检查（示例）
- runtime-state.json - 运行时状态（示例）
